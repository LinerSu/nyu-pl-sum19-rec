```
brew install smlnj
```
Once installed, type `sml` in your terminal.

### Exercise
1. Write an regular expression that matches the positive float point number with the following restriction:
```
Match:
1.2
0.35
0.007
0.0
Not match:
+1.2
-3.4
01.23
3
0
```

**Solution:**
    <p>

```
([1-9][0-9]*|0)\.[0-9]+
```
   </p>
